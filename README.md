Pada project mobile programming ini, kami membuat aplikasi yang bernama FruitLite. FruitLite adalah sebuah aplikasi berbasis android yang bertujuan untuk membantu user untuk lebih mengetahui detail dari buah buahan.

Berikut ini adalah dokumentasi dari aplikasi kami. 

Halaman Register
Pada halaman register, user bisa mendaftarkan diri mereka di dalam aplikasi dengan memasukkan username, email, password dan confirm password.

<img width="206" alt="register" src="https://user-images.githubusercontent.com/84964642/149962417-563b23be-111b-48f1-bd4e-41d5bdead625.png">

Halaman Login
Pada halaman login, user memasukkan email dan password mereka untuk masuk ke dalam aplikasi.

<img width="206" alt="Login" src="https://user-images.githubusercontent.com/84964642/150111683-487b24ef-8468-46d3-9607-c8ce80ac409b.png">

Halaman Home
Pada halaman home, user bisa melihat list buah-buahan. Apabila user mengklik salah satu dari buah tersebut, maka user akan bisa melihat penjelasan detail dari buah tersebut.
